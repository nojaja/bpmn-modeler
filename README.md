# bpmn-modeler
A bpmn modeler that embeds the bpmn.io project and supports Google Drive

## Feature
* Edit the BPMN 2.0 diagram in the browser.
* Google Drive supported.
* BPMN 2.0 diagrams supported.
* Drag and drop to import images.
* NFS supported.
